# keyforge-sync

Syncs keyforge decks between mastervault and other keyforce sites like Decks of Keyforge and The Cruicble

**_USE THIS AT YOUR OWN RISK_**

Seriously. I make no promises that any one of these sites won't shut down your account, ban you, or any other type of
stuff.
